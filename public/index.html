Frequently asked questions

Common questions before checkout

Can I filter by a specific country?

Yes. Search by country name or ISO code and the app requests plans for that destination.

How does ordering work?

Pick a plan, confirm the quantity, and the backend sends a secure order request to Roamify.

Does this expose the API token?

No. The token stays on the server in `.env`, and the frontend only calls local Express routes.

What if a plan includes throttling?

The card will show the throttle note when Roamify marks the package as throttled.
